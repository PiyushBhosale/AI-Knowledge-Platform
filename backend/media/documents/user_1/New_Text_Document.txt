## PHASE 1 — FOUNDATION (Months 1-6)

### 1. DSA & Problem Solving (3 months)

**Week 1-2: Arrays & Strings**
- Two pointer technique
- Sliding window (fixed & variable)
- Prefix sum
- Kadane's algorithm
- Dutch national flag problem
- String manipulation — reversal, anagram check, palindrome
- Practice: 15-20 Leetcode easys

**Week 3-4: Hashing**
- HashMap internals (how hashing works, collision handling)
- Frequency counting
- Two sum patterns
- Group anagrams
- Subarray sum equals K
- Practice: 10-15 problems

**Week 5-6: Linked Lists**
- Singly, doubly, circular
- Fast & slow pointer (Floyd's cycle detection)
- Reverse a linked list (iterative + recursive)
- Merge two sorted lists
- LRU Cache implementation
- Practice: 10-12 problems

**Week 7-8: Stacks & Queues**
- Stack using array & linked list
- Monotonic stack
- Next greater element
- Valid parentheses & variants
- Min stack
- Queue using stacks
- Sliding window maximum (deque)
- Practice: 12-15 problems

**Week 9-10: Trees**
- Binary tree traversals (inorder, preorder, postorder — iterative + recursive)
- Level order traversal (BFS)
- Height, diameter, balanced check
- Lowest Common Ancestor
- Binary Search Tree — insert, delete, validate
- Serialize/deserialize
- Practice: 20 problems

**Week 11: Heaps & Priority Queues**
- Min heap, max heap internals
- Heap sort
- Top K elements pattern
- Merge K sorted lists
- Median from data stream
- Practice: 8-10 problems

**Week 12: Graphs**
- Adjacency list & matrix representation
- BFS & DFS (iterative + recursive)
- Connected components
- Cycle detection (directed & undirected)
- Topological sort (Kahn's + DFS)
- Dijkstra's shortest path
- Union Find (disjoint set)
- Number of islands, course schedule, clone graph
- Practice: 20 problems

**Week 13: Dynamic Programming**
- Memoization vs tabulation
- 1D DP — climbing stairs, house robber, coin change
- 2D DP — longest common subsequence, edit distance, unique paths
- Knapsack (0/1 and unbounded)
- Longest increasing subsequence
- DP on strings
- Practice: 20-25 problems

**Resources:**
- Neetcode 150 (primary list)
- Striver's A2Z DSA sheet (supplement)
- Abdul Bari on YouTube for theory
- Do NOT just watch — solve then watch explanation

## THE NON-NEGOTIABLE RULES

1. **Build projects for every phase** — knowledge without projects is worthless in interviews
2. **Never stay at a company more than 2 years** until you hit target
3. **Interview even when you're not looking** — keeps you sharp
4. **Network on LinkedIn and Twitter** — cold DMs work, people help
5. **Track progress weekly** — what you learned, what you built, what's next
6. **Sleep properly** — burnout kills more careers than lack of skill

This is the whole map. Execute it.
